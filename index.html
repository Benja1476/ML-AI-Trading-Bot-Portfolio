Project Overview

This project implements a professional Reinforcement Learning (PPO) based trading bot for crypto markets. The bot is mobile-accessible and runs on server, providing real-time trading, logging, and Telegram notifications.

  • Data: Historical OHLCV from Binance or CSV
  • Indicators: RSI, EMA55
  • Environment: Custom Gym TradingEnv (13 features)
  • Model: PPO (Stable Baselines3)
  • Deployment: Async Python bot on server, mobile-friendly

Tech Stack & Tools

  • Python 3.11+, asyncio, ccxt, ta
  • Stable-Baselines3 (PPO)
  • Pandas, NumPy for data processing
  • Telegram Bot API for notifications
  • Gymnasium for environment simulation
  • Hosted on VPS / Mobile-friendly server
